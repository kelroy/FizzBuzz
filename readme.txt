#Program FizBuzz
#Author Andrew Koerner

	This program prints the numbers from 1 to 100.  For numbers that are multiples of three Fizz is printed to the console.  For numbers that are multiples of five Buzz is printed to the console.
For numbers that are multiples of both FizzBuzz is output to the console.

To run type "ruby FizzBuzz.rb" into the console.
